# 组件说明

## 8250串口

本组件用于实现8250串口及其兼容型号（如16550）的相关辅助代码。

8250串口兼容外设广泛运用于各种SOC中。

使用串口可方便打开一个控制台，进行基础交互。

在[Qemu](https://www.qemu.org/)的非x86平台模拟中，无论是运行操作系统（如Linux），还是直接跑裸机代码，串口是最基础的交互方式。
