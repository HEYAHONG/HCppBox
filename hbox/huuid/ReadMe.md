# 说明

本目录用于实现`UUID`相关代码，`UUID`常用作唯一标识。

# `UUID`版本

`UUID`根据唯一标识的生成算法和适用场景可区分不同的版本。

| 版本 | 说明                                            | 备注                   |
| ---- | ----------------------------------------------- | ---------------------- |
| `V1` | 基于时间戳 + MAC 地址                           |                        |
| `V2` | 基于 DCE 安全                                   | 已废弃，本组件尚未实现 |
| `V3` | 基于 MD5 哈希                                   |                        |
| `V4` | 基于随机数                                      |                        |
| `V5` | 基于 SHA-1 哈希                                 |                        |
| `V6` | 基于时间戳 +MAC 地址,重新排列了`V1`版本的UUID。 |                        |
| `V7` | 基于系统时间戳+随机数                           |                        |
| `V8` | 用户自定义                                      |                        |

# 短UUID

在某些场景中（如蓝牙）下，需要缩短`UUID`。

本组件支持以下版本：

- 16位UUID:128位UUID第一部分的低16位。
- 32位UUID:128位UUID第一部分(32位)。
