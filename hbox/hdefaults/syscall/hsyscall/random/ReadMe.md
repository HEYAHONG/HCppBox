# 说明

随机数相关系统调用。

默认情况下，生成伪随机数(使用`hcrypto`组件的随机数生成算法)。

# 外部配置的宏定义

可外部配置的宏定义如下:

| 宏定义                    | 说明                  | 备注                                  |
| ------------------------- | --------------------- | ------------------------------------- |
| `HSYSCALL_GETRANDOM_TINY` | 使用精简版`getrandom` | 精简版`getrandom`不可用于加密类应用。 |
