# 说明

随机数相关系统调用。

默认情况下，生成伪随机数(使用`hcrypto`组件的随机数生成算法)。

# 外部配置的宏定义

可外部配置的宏定义如下:

| 宏定义                        | 说明                             | 备注                                                       |
| ----------------------------- | -------------------------------- | ---------------------------------------------------------- |
| `HSYSCALL_GETRANDOM_TINY`     | 使用精简版`getrandom`            | 精简版`getrandom`不可用于加密类应用。                      |
| `HSYSCALL_GETRANDOM_USING_MT` | 使用梅森旋转法生成不安全的随机数 | 梅森旋转法将消耗约2KB内存，在RAM容量较小的设备不推荐使用。 |
