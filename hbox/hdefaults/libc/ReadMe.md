# 说明

此目录用于对C运行库的部分函数进行包装。

# 目录说明

- [wrapper](wrapper)：包装C运行库库函数

# C运行库

## `putchar`

将字符输出到标准输出上。

可外部配置的宏定义如下:

| 宏定义     | 说明                | 备注                     |
| ---------- | ------------------- | ------------------------ |
| `HPUTCHAR` | 用户实现的`putchar` | 参数与返回值同`hputchar` |

## `getchar`

从标准输入上读取字符。

可外部配置的宏定义如下:

| 宏定义     | 说明                | 备注                     |
| ---------- | ------------------- | ------------------------ |
| `HGETCHAR` | 用户实现的`getchar` | 参数与返回值同`hgetchar` |
