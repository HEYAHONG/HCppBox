# 说明

时间相关实现。

注意：与标准C语言库实现相比，默认时区为东八区，无夏令时。

# 外部配置的宏定义

可外部配置的宏定义如下:

| 宏定义                          | 说明                               | 备注                       |
| ------------------------------- | ---------------------------------- | -------------------------- |
| `HLIBC_TIME_TIMEZONE_INFO_HOOK` | 宏函数，参数与返回值均为时区信息。 | 如不定义则使用全局时区信息 |
