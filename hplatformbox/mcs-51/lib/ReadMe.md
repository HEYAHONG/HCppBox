# 说明

MCS-51 公共代码库。

本库中除[libmono.c](libmono.c)的其他文件均由[libmono.c](libmono.c)包含，无需用户手动包含。

# 使用

虽然SDCC支持多个文件链接，但本库使用单文件方式，可通过以下方式在主程序C源代码中使用：

```c
/*
 * 头文件
 */

/*
 * 变量,函数声明
 */

/*
 * libmono.c 的配置宏定义
 */

/*
 * 包含libmono.c,具体路径应根据实际路径与包含路径设置确定
 */

#include "libmono.c"

/*
 * 实际业务代码(包括main函数)
 */

```
